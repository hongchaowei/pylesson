数据使用指南

🚀 快速开始

# 1. 安装必要的库 pip install pandas numpy matplotlib seaborn yfinance # 2. 加载数据 import pandas as pd import numpy as np # 加载股价数据 df = pd.read_csv('stock_prices.csv', parse_dates=['date']) print(df.info()) # 3. 基础分析 # 计算收益率 df['returns'] = df.groupby('symbol')['close'].pct_change() # 计算移动平均 df['ma_20'] = df.groupby('symbol')['close'].rolling(20).mean() # 4. 可视化 import matplotlib.pyplot as plt plt.figure(figsize=(12, 6)) plt.plot(df[df['symbol']=='000001']['date'], df[df['symbol']=='000001']['close']) plt.title('股价走势图') plt.show()

📋 数据字段说明

股价数据字段
  • date: 交易日期 (YYYY-MM-DD)
  • symbol: 股票代码 (6位数字)
  • open/high/low/close: 开高低收价格
  • volume: 成交量 (股)
  • amount: 成交额 (元)
  • adj_close: 复权收盘价

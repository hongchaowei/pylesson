什么是Python？

Python是一种高级、解释型、通用编程语言，由Guido van Rossum于1991年首次发布。Python的设计哲学强调代码的可读性和简洁性，使用缩进而非括号来划分代码块。它是一种动态类型语言，支持多种编程范式，包括面向对象、命令式、函数式和过程式编程。

Python的主要特点

  • 易于学习和使用：Python的语法简洁明了，接近自然语言，降低了学习门槛
  • 可读性强：Python代码结构清晰，易于理解和维护
  • 丰富的库和框架：Python拥有大量的标准库和第三方库，可以满足各种应用需求
  • 跨平台：Python可以在Windows、macOS、Linux等多种操作系统上运行
  • 解释型语言：Python代码无需编译，可以直接运行，便于调试和测试
  • 强大的社区支持：Python拥有庞大的用户社区，提供丰富的资源和支持

Python版本

目前，Python有两个主要的版本：

  • Python 2.x：已于2020年1月1日停止支持，不再接收安全更新
  • Python 3.x：当前活跃开发的版本，引入了许多改进和新特性

在本课程中，我们将使用Python 3，因为它是Python的未来，并且提供了更好的语言特性和库支持。

安装Python

在开始编写Python程序之前，我们需要在计算机上安装Python解释器。以下是在不同操作系统上安装Python的步骤：

Windows系统

  1. 访问Python官方网站：https://www.python.org/downloads/
  2. 下载最新版本的Python安装程序（选择Windows版本）
  3. 运行安装程序，勾选"Add Python to PATH"选项
  4. 点击"Install Now"开始安装
  5. 安装完成后，打开命令提示符（CMD）或PowerShell
  6. 输入python --version验证安装是否成功

macOS系统

  1. macOS通常预装了Python 2.x版本，但我们需要安装Python 3.x
  2. 推荐使用Homebrew安装Python：
    • 首先安装Homebrew（如果尚未安装）：
      /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
    • 然后安装Python 3：
      brew install python3
  3. 或者，从Python官方网站下载macOS安装程序
  4. 安装完成后，打开终端
  5. 输入python3 --version验证安装是否成功

Linux系统

  1. 大多数Linux发行版已预装Python，但可能需要更新到最新版本
  2. 对于Ubuntu/Debian系统：
    • sudo apt update
    • sudo apt install python3 python3-pip
  3. 对于Fedora/RHEL/CentOS系统：
    • sudo dnf install python3 python3-pip
  4. 安装完成后，打开终端
  5. 输入python3 --version验证安装是否成功

设置开发环境

安装Python后，我们需要设置一个舒适的开发环境。以下是一些常用的Python开发工具：

集成开发环境(IDE)

IDE提供了代码编辑、调试、测试等综合功能，适合大型项目开发。

  • PyCharm：功能强大的Python IDE，提供社区版（免费）和专业版
    • 下载地址：https://www.jetbrains.com/pycharm/download/
    • 提供代码补全、调试、测试、版本控制等功能
  • Visual Studio Code (VS Code)：轻量级但功能强大的编辑器，通过插件支持Python
    • 下载地址：https://code.visualstudio.com/
    • 安装Python扩展：在VS Code中，按Ctrl+Shift+X打开扩展面板，搜索"Python"并安装

Jupyter Notebook

Jupyter Notebook是一个交互式计算环境，特别适合数据分析和可视化，在金融领域广泛使用。

  1. 安装Jupyter Notebook：
    • 打开命令行/终端
    • 运行：pip install jupyter
  2. 启动Jupyter Notebook：
    • 在命令行/终端中，导航到您想要保存笔记本的目录
    • 运行：jupyter notebook
    • 这将在您的默认浏览器中打开Jupyter Notebook界面
  3. 创建新的笔记本：
    • 在Jupyter界面中，点击"New"按钮，然后选择"Python 3"
    • 这将创建一个新的Python笔记本，您可以在其中编写和执行代码

Anaconda发行版

Anaconda是一个Python发行版，包含了Python解释器、Jupyter Notebook和许多常用的科学计算和数据分析库，特别适合金融应用开发。

  1. 下载Anaconda：https://www.anaconda.com/products/individual
  2. 安装Anaconda：
    • 运行下载的安装程序
    • 按照安装向导的指示完成安装
  3. 使用Anaconda Navigator：
    • 安装完成后，启动Anaconda Navigator
    • 从Navigator中，您可以启动Jupyter Notebook、Spyder IDE和其他工具

Python交互式环境

Python提供了一个交互式环境，允许您直接输入和执行Python代码。这对于学习和测试小段代码非常有用。

使用Python解释器

  1. 打开命令行/终端
  2. 输入python（Windows）或python3（macOS/Linux）启动Python解释器
  3. 您将看到Python版本信息和提示符（>>>），表示解释器已准备好接受输入
  4. 尝试输入一些简单的Python命令： 
    >>> print("Hello, Python!")
Hello, Python!
>>> 2 + 3
5
>>> name = "Python"
>>> print("Hello, " + name + "!")
Hello, Python!
  5. 要退出解释器，输入exit()或按Ctrl+Z（Windows）或Ctrl+D（macOS/Linux）

使用Jupyter Notebook

Jupyter Notebook提供了一个更丰富的交互式环境，特别适合数据分析和可视化。

  1. 启动Jupyter Notebook（如前所述）
  2. 创建一个新的Python笔记本
  3. 在代码单元格中输入Python代码，然后按Shift+Enter执行： 
    # 这是一个简单的Python代码示例
print("Hello, Jupyter!")
# 进行一些计算
result = 0
for i in range(1, 11):
    result += i
print("1到10的和是:", result)
# 创建一个简单的图表（需要matplotlib）
import matplotlib.pyplot as plt
plt.figure(figsize=(8, 4))
plt.plot([1, 2, 3, 4, 5], [1, 4, 9, 16, 25], 'ro-')
plt.title('简单的平方函数图')
plt.xlabel('x')
plt.ylabel('x²')
plt.grid(True)
plt.show()

第一个Python程序

现在，让我们创建并运行一个简单的Python程序，这将是您Python编程之旅的第一步。

创建Python脚本文件

  1. 打开您选择的文本编辑器或IDE
  2. 创建一个新文件，命名为hello.py
  3. 输入以下代码： 
    # 这是我的第一个Python程序
print("欢迎来到Python金融编程课程!")
# 获取用户输入
name = input("请输入您的姓名: ")
print("您好，" + name + "！很高兴认识您。")
# 进行简单的金融计算
principal = 1000  # 本金
rate = 0.05       # 年利率
years = 5         # 年数
# 计算复利
amount = principal * (1 + rate) ** years
interest = amount - principal
print(f"如果您投资{principal}元，年利率为{rate*100}%，")
print(f"{years}年后，您将获得{amount:.2f}元，其中利息为{interest:.2f}元。")
  4. 保存文件

运行Python脚本

  1. 打开命令行/终端
  2. 导航到保存hello.py文件的目录
  3. 运行脚本：
    • Windows：python hello.py
    • macOS/Linux：python3 hello.py
  4. 按照提示输入您的姓名，然后查看输出结果

代码解释

  • # 这是我的第一个Python程序：这是一个注释，不会被执行
  • print("欢迎来到Python金融编程课程!")：显示欢迎消息
  • name = input("请输入您的姓名: ")：获取用户输入并存储在变量name中
  • print("您好，" + name + "！很高兴认识您。")：显示个性化问候
  • 接下来的几行代码演示了一个简单的金融计算：计算复利
  • f"..."：这是一个f-string（格式化字符串），允许在字符串中嵌入变量值
  • {amount:.2f}：格式化数字，保留两位小数

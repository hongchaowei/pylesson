第一部分 - Python基础

模块4: 面向对象编程

掌握面向对象编程思想，构建可维护的金融应用

模块概述

在这个模块中，你将学习Python的面向对象编程概念，这是一种强大的编程范式，可以帮助你更好地组织和管理代码。我们将探讨如何使用这些概念来构建金融应用程序。

封装性

将数据和方法封装在类中，提高代码的安全性和可维护性

继承性

通过继承机制实现代码重用，构建类的层次结构

多态性

同一接口可以有不同的实现，提高代码的灵活性

设计模式

学习常见的设计模式，解决金融应用中的典型问题

课程内容

课程1: 类与对象

学习如何定义类、创建对象，以及如何使用属性和方法。通过实际的金融应用示例，如股票类和投资组合类，来理解面向对象编程的基础概念。

开始学习

课程2: 继承与多态

探索类的继承机制、方法重写和多态性。学习如何创建类的层次结构，以及如何在金融应用中使用这些概念来模型化不同类型的金融工具。

开始学习

课程3: 特殊方法和属性

学习Python的特殊方法（魔术方法）和属性装饰器。了解如何自定义对象的行为，实现运算符重载，以及如何在金融应用中应用这些概念。

开始学习

课程4: 设计模式

了解常见的面向对象设计模式，如单例模式、工厂模式、观察者模式和策略模式。学习如何在金融应用中应用这些模式来解决常见的设计问题。

开始学习

预备知识

在开始本模块之前，你应该已经：

  • 完成了前三个模块的学习
  • 熟悉Python的基本语法和数据类型
  • 了解函数的定义和使用
  • 掌握基本的程序控制流程
